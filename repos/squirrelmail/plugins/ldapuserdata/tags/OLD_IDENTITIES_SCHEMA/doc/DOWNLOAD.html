CVS Repository

You can grab the latest sources via anonymous CVS, using this command sequence: (Press enter when asked for a password)

cvs -d:pserver:anoncvs@email.uoa.gr:/squirrelmail login
cvs -z3 -d:pserver:anoncvs@email.uoa.gr:/squirrelmail co ldapuserdata

There is also a ViewCVS Web Interface to the repository.
