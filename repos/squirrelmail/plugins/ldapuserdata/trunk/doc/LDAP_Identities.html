LDAP Identities

Before moving on and enabling the LDAP-based Identities feature of this plugin, please read some information about the Repfilter Sendmail Milter and its design.

If you'd like to use LDAP identities, then you can try using the supplied Personal Options patch. This will patch the file include/options/personal.php and change the link that points to Squirrelmail's "Advanced Identities", to the one provided by this plugin.

The new identities screen will then display the available email addresses from LDAP, as described in this design.

This patch and LDAP Identities scheme originates from a customized environment, intended for the University of Athens Email Project. It released in the hope that it could be useful to someone. It makes a fine complement to the Repfilter Sendmail Milter.
