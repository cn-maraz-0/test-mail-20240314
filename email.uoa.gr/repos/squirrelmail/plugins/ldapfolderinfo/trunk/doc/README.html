ldapfolderinfo Squirrelmail Plugin

Introduction

ldapfolderinfo is a Squirrelmail plugin and patchset that is essential to the email.uoa.gr Mail Project. Based on theCyrus patches and Cyrusmaster Cyrus administration tool, it provides a wealth of information about mail folders and their users and administrators to end users. It also allows them to easily post to a folder through the webmail interface.

ldapfolderinfo was originally based on folder_sizes v0.4 by Robin Rainton, robin @ rainton.com, http://rainton.com/fun/freeware">

Important Note (Read this Before you Move on)

This plugin was written to suit the needs of a customized LDAP-based environment. It is released as open-source, GPLed code, in the hope that it (or some parts of it) will be useful to someone. Usual GPL disclaimer applies.

Additionally:

  • The Cyrusmaster and Cyrus patches are not yet documented properly.
  • This is a bit intrusive plugin, because some original Squirrelmail files need to be patched.
  • The code can be considered beta, and it is not yet polished in all places.
