Υπηρεσία Καταλόγου

Αναζήτηση

Για να αναζητήσετε ένα μέλος της Πανεπιστημιακής Κοινότητας, μπορείτε να εισάγετε τα απαιτούμενα κριτήρια στη φόρμα αναζήτησης του πάνω πλαισίου. Όταν επιλέξετε "Αναζήτηση" ή πατήσετε το πλήκτρο Return, τα αποτελέσματα της αναζήτησης θα φανούν στο κάτω πλαίσιο.

Υπάρχουν δύο τρόποι αναζήτησης: ο απλός και αυτός για προχωρημένους. Στην απλή αναζήτηση, ό,τι εισάγετε θα αναζητηθεί στο όνομα, επώνυμο και email διεύθυνση των χρηστών του Πανεπιστημίου. Στη δεύτερη περίπτωση, μπορείτε να ορίσετε με λεπτομέρεια τα εξής κριτήρια:

  • Όνομα
  • Επίθετο
  • Διεύθυνση Email
  • Ιδιότητα (Μέλος Δ.Ε.Π., Προσωπικό, Φοιτητής κ.λπ.)
  • Τίτλος (Υποκατηγορία της Ιδιότητας, όπως Επίκουρος Καθηγητής, Προπτυχιακός Φοιτητής κ.λπ.)

Περιήγηση

Επιλέγοντας την "Περιήγηση" ως τρόπο λειτουργίας, μπορείτε να δείτε εποπτικά όλες τις οργανωτικές μονάδες του Πανεπιστημίου Αθηνών. Όταν επιλέξετε κάποια Μονάδα, έχετε δύο επιλογές:

  • Περιήγηση σε όλους τους ανθρώπους αυτής της οργανωτικής μονάδας. Η επιλογή αυτή μάλλον θα φέρει πολλά αποτελέσματα και δε θα είναι ιδιαίτερα χρήσιμη για μεγάλα τμήματα του Πανεπιστημίου.
  • Αναζήτηση σε ανθρώπους αυτής της οργανωτικής μονάδας. Μπορείτε έτσι να ορίσετε περαιτέρω κριτήρια, όπως όνομα ή ιδιότητα, για μία συγκεκριμένη οργανωτική μονάδα του Πανεπιστημίου.

Χρήσιμες Συμβουλές

  • Όταν εισάγετε κάποιο όνομα, δεν παίζει ρόλο η εισαγωγή μικρών ή κεφαλαίων χαρακτήρων. Όμως οι τόνοι θα πρέπει να εισάγονται σωστά σε όλες τις περιπτώσεις.
  • Καλό είναι να ορίσετε κατά την αναζήτησή σας την Ιδιότητα του ανθρώπου που αναζητείτε, ώστε να μειώσετε τον αριθμό αποτελεσμάτων.
  • Στα αποτελέσματα της αναζήτησης μπορείτε να πατήσετε το εικονίδιο για να δείτε τα αποτελέσματα της αναζήτησης σε όλο το παράθυρο του browser.

Για τα μέλη της Πανεπιστημιακής Κοινότητας

Τα μέλη της Πανεπιστημιακής Κοινότητας που έχουν λογαριασμό στο Κέντρο Λειτουργίας και Διαχείρισης Δικτύου μπορούν να επεξεργαστούν μερικά από τα επιπλέον στοιχεία τους που θα φαίνονται στην Υπηρεσία Καταλόγου, μέσω της υπηρεσίας Webmail.

Αν επιθυμείτε να αλλάξει κάποιο από τα βασικά στοιχεία που απεικονίζονται στον κατάλογο, θα πρέπει να κάνετε μία ηλεκτρονική αίτηση αλλαγής στοιχείων. Για περισσότερες πληροφορίες, δείτε τις Σελίδες Υποστήριξης Χρηστών.
